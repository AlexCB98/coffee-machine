# Coffee Machine 

A simple Coffee Machine program made in Python.

This project was built as part of my Python learning journey from Angela Yu's 100 Days of Code course.

The program simulates a coffee machine where the user can:
- choose a drink
- insert coins
- receive change
- consume machine resources
- view the current machine report

---

## Features

- Espresso, Latte and Cappuccino
- Resource management system
- Coin processing system
- Transaction checking
- Change calculation
- Dynamic ingredient validation
- Error handling with `try/except`
- Report system
